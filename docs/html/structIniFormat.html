Detailed Description

24-bit bitfield representing a unique format of an INI file (INI dialect) – sizeof(IniFormat) should be 3

Structure of the bitfield:

  • Bits 1-18: INI syntax
  • Bits 19-20: INI semantics
  • Bits 21-22: user's syntax (disabled entries)
  • Bits 23-24: unused
