size_t ini_collapse_array ( char *const  ini_string,
const char  delimiter,
const IniFormat  format 
)

Removes spaces around all the delimiters of a stringified array.

Parameters
ini_stringThe stringified array
delimiterThe delimiter of the array members
formatThe format of the INI file
Returns
The new length of the string containing the array

Out of quotes similar to ECMAScript ini_string.replace(new RegExp("^\\s+|\\s*(?:(" + delimiter + ")\\s*|($))", "g"), "$1$2"). If INI_ANY_SPACE (0) is used as delimiter one or more different spaces (/[\t \v\f\n\r]+/) will always be collapsed to one space (' '), independently of their position.

Usually ini_string comes from an IniDispatch, but any other string can be used as well.
